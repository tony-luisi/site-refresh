
# site-refresh

its my personal website

## how

### install

```
git clone https://github.com/tony-luisi/site-refresh

cd

npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy

```

### relevant skills (key development, accounting and personal skills)
